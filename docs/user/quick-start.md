# 快速上手

本文主要介绍最常用的白板操作（书写/擦除/选择、撤销恢复、缩放平移、多页面）。

## 界面速览

- 底部中央工具栏：**选择 / 书写 / 擦除**、**撤销 / 恢复**。
- 左下角：**更多**（设置/导出/退出）、**最小化**、**导入**。
- 右下角：**分页控制条**（上一页/下一页/新增页、页码与页面列表）。
- 可选：如果启用了「视频展台」，底部中央右侧会出现 **视频展台** 按钮。

## 书写与画笔设置

- 点击 **书写** 进入书写模式。
- 再次点击 **书写** 会弹出「画笔设置」：
  - 粗细：3 档
  - 颜色：9 种预设色
- 支持触控、手写笔、鼠标（鼠标需要按住左键）。

## 擦除与滑动清屏

- 点击 **擦除** 进入橡皮擦模式。
- 再次点击 **擦除** 会弹出「清屏滑块」：
  - 向右滑到阈值后松开即可清空画布
  - 清空后会自动切回 **书写**

## 选择与编辑（笔迹 / 附件）

- 点击 **选择** 进入选择模式。
- 笔迹选择：使用 InkCanvas 的选区操作选中笔迹后，会出现悬浮 Dock（置顶/复制/删除）。
- 附件选择：在选择模式下点击附件卡片即可选中，出现选中框与缩放手柄。
  - **拖动**选中框可移动
  - **拖动手柄**可调整尺寸
  - 悬浮 Dock 的 **复制** 目前仅对笔迹生效，附件不支持复制

## 撤销 / 恢复

- 点击底部工具栏的 **撤销 / 恢复**。
- 也可尝试使用快捷键（取决于焦点是否在画布上）：`Ctrl+Z` 撤销、`Ctrl+Y` 恢复。

## 缩放与平移

- 鼠标滚轮：以光标所在位置为中心缩放。
- 鼠标右键拖动：平移画布视图。
- 触控：双指缩放/平移（可在设置中限制为“仅双指手势”）。

## 多页面

- 右下角分页控制条：
  - **上一页 / 下一页**
  - **新增页**
  - 点击页码/指示器：打开页面列表并进行切换/删除（带缩略预览）
