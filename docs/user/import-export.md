# 导入、附件与导出

本文介绍导入（图片/视频/文本/链接/WBI）、附件的编辑方式，以及导出 PNG/JPG/PDF/WBI 的用法与限制。

## 导入

点击左下角 **导入**，在弹窗中选择不同类型：

### 图片

- 选择图片文件后导入到当前页面。
- 图片会以卡片形式出现在画布中；图片解码在后台进行，导入后可能先显示占位信息，稍后显示预览。

### 视频

- 选择视频文件后导入到当前页面（以卡片占位显示）。
- 在**选择模式**下双击视频卡片，可用系统默认方式打开外部播放器。

### 文本

- 支持直接粘贴/输入文本导入。
- 也支持从文本文件导入（会读取文件内容后生成文本卡片）。

### 链接

- 支持输入或从剪贴板粘贴 URL。
- 未写协议时会尝试自动补全为 `https://`，仅接受 `http/https`。
- 在**选择模式**下双击链接卡片，会用默认浏览器打开。

### WBI 文件（WindBoard 专用格式）

- 选择 `.wbi` 后会显示文件信息，并可选择导入模式：
  - **追加到现有页面之后**
  - **替换所有现有页面**
- 导入完成后：
  - 会恢复导出时的画布状态（笔迹、附件位置与大小、视图缩放/平移等）
  - 若存在缺失资源（例如视频路径失效、未打包的图片路径失效），会提示列表；缺失的附件会以占位形式显示

## 附件编辑（选择模式）

1. 点击底部 **选择** 进入选择模式。
2. 点击附件卡片选中：
   - 拖动选中框移动
   - 拖动边/角手柄缩放
3. 使用悬浮 Dock：
   - **置顶 / 取消置顶**：切换附件层级（置顶后会显示在笔迹上方）
   - **删除**：删除附件
   - **复制**：目前仅对笔迹生效（附件不支持复制）

## 导出

点击左下角 **更多** → **导出**，在弹窗中选择导出范围与格式：

### 图片（PNG/JPG）

- 单页：导出到指定文件。
- 多页：导出到同一文件夹，按前缀+序号命名。

### PDF

- 支持横向/纵向、缩放模式、DPI 等选项。

### WBI（WindBoard 专用格式）

- WBI 用于“可还原”的导出：页面尺寸、笔迹、附件、视图缩放/平移等会被保留。
- **包含图片附件** 开启时：会把图片资源打包进 WBI（导入时不依赖原始路径；资源会被压缩以减小体积）。
- 视频与链接：仅保存路径/URL（导入时如果路径不存在，会提示缺失）。
